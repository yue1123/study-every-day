## 验证回文串

给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。
说明：本题中，我们将空字符串定义为有效的回文串。

## 示例 1:

```txt
输入: "A man, a plan, a canal: Panama"
输出: true
解释："amanaplanacanalpanama" 是回文串
```

## 示例 2:

```txt
输入: "race a car"
输出: false
解释："raceacar" 不是回文串
```

## 提示：

```txt
1 <= s.length <= 2 * 105
字符串 s 由 ASCII 字符组成
```

# 题解

双指针,一个往前,一个往后,遇到特殊字符就跳过,判断每次是否相等

# 常用字符串编码

> 0 -- 9 ： 48 -- 57
> A -- Z ： 65 -- 90
> a -- z ： 97 -- 122
