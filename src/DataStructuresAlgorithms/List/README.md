# 列表

列表是一种数据项构成的有限序列，即按照一定的线性顺序，排列而成的数据项的集合，在这种数据结构上进行的基本操作包括对元素的的查找，插入，和删除

# 抽象数据类型定义

## 属性

| 属性名    | 描述             |
| --------- | ---------------- |
| pos       | 当前列表位置     |
| dataStore | 列表数据存储仓库 |
| listSize  | 列表元素个数     |
| length    | 返回列表元素个数 |

## 方法

| 方法名     | 描述                         |
| ---------- | ---------------------------- |
| append     | 列表末尾添加元素             |
| insert     | 列表中插入元素               |
| getElement | 获取列表元素                 |
| remove     | 移除列表某个元素             |
| clear      | 清空列表元素                 |
| moveTo     | 将当前列表位置移动到指定位置 |
| next       | 列表位置右移动               |
| prev       | 列表位置左移动               |
| front      | 列表位置移动到第一个元素     |
| end        | 列表位置移动到最后一个元素   |
| currPos    | 返回列表当前元素             |

注意:

```js
let i = 10
// 前置加加可以看成 "同步",取到的是+1后的值
++i === 10   // true

// 后置加加可以看做是 "异步",取到的是没有+1的值,当前语句执行完了才会+1
i++ === 11  // true
```
